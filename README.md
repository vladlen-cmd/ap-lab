# Android Application Development Lab (TEE7111)

1. **[Practical 1](./Practical-1-Development-Setup): Setting Up Android Development Environment**
   Aim: Install and configure Android Studio, then understand the structure of an Android project.

2. **[Practical 2](./Practical-2-Hello-World): Hello World Application**
   Aim: Create and run a basic Android application that displays a simple message.

3. **[Practical 3](./Practical-3-Activity-Lifecycle): Android Components (Activities & Lifecycle)**
   Aim: Study Android activities and understand activity lifecycle methods.

4. **[Practical 4](./Practical-4-Android-Services): Implementing Services**
   Aim: Create and manage background services in an Android application.

5. **[Practical 5](./Practical-5-Content-Provider): Content Providers**
   Aim: Understand and implement content providers for data sharing between applications.

6. **[Practical 6](./Practical-6-Broadcast-Receiver): Broadcast Receivers**
   Aim: Implement broadcast receivers to handle system-wide events.

7. **[Practical 7](./Practical-7-UI-Design): User Interface Design**
   Aim: Design user interfaces using various layouts and UI components.

8. **[Practical 8](./Practical-8-Dialog-Pickers): Dialogs and Pickers**
   Aim: Implement dialogs and picker controls for user interaction.

9. **[Practical 9](./Practical-9-Intents-Usage): Intents**
   Aim: Understand and implement explicit and implicit intents for activity navigation and communication.

10. **[Practical 10](./Practical-10-Notifications-System): Notifications**
    Aim: Create and display notifications in an Android application.

11. **[Practical 11](./Practical-11-SQLite-Basics): SQLite Database (CRUD)**
    Aim: Perform create, read, update, and delete operations using an SQLite database.

12. **[Practical 12](./Practical-12-Database-Connection): Database Connectivity**
    Aim: Connect Android applications with an SQLite database and perform data operations.
