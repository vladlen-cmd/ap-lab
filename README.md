# Android Application Development Lab (TEE7111)  
## Aims of Practicals 1–12

1. **Practical 1: Setting Up Android Development Environment**  
   Aim: To install and configure Android Studio and understand the structure of an Android project.

2. **Practical 2: Hello World Application**  
   Aim: To create and run a basic Android application displaying a simple message.

3. **Practical 3: Android Components (Activities & Lifecycle)**  
   Aim: To study Android activities and understand the activity lifecycle methods.

4. **Practical 4: Implementing Services**  
   Aim: To create and manage background services in an Android application.

5. **Practical 5: Content Providers**  
   Aim: To understand and implement content providers for data sharing between applications.

6. **Practical 6: Broadcast Receivers**  
   Aim: To implement broadcast receivers to handle system-wide events.

7. **Practical 7: User Interface Design**  
   Aim: To design user interfaces using various layouts and UI components.

8. **Practical 8: Dialogs and Pickers**  
   Aim: To implement dialogs and picker controls for user interaction.

9. **Practical 9: Intents**  
   Aim: To understand and implement explicit and implicit intents for activity navigation and communication.

10. **Practical 10: Notifications**  
    Aim: To create and display notifications in an Android application.

11. **Practical 11: SQLite Database (CRUD)**  
    Aim: To perform create, read, update, and delete operations using SQLite database.

12. **Practical 12: Database Connectivity**  
    Aim: To connect Android applications with SQLite database and perform data operations.
